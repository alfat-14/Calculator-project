# Python Calculator

A simple calculator built using Python. This program performs basic arithmetic operations and runs in a loop until the user exits.

## Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Modulo (%)
- Handles division/modulo by zero
- Input validation
- Continuous loop until exit

## How to Run
1. Make sure Python is installed
2. Run the file:
   python calculator.py

## Example
Enter first number: 10  
Enter second number: 5  
Operator: +  
Result: 15
